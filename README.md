### koa-booster-cli
- Usage
  - Step 1
  ```
  npm i -g koa-booster-cli
  ```
  - Step 2
  ```
  koa-booster create {projectName}

  ```
  或者
  ```
  mkdir {projectName} && cd {projectName}
  koa-booster create .
  ```
  - Step 3 

  ```
  cd {projectName} && npm i
  ```
- Related Doc [koa-booster](https://www.npmjs.com/package/koa-booster)
